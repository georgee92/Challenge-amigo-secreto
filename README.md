Amigo Secreto

Descipción Este proyecto es una página web interactiva desarrollada en JavaScript, HTML y CSS, permite crear una lista de amigos y a partir de ahí, gestionar un sorteo del "Amigo Secreto".

El objetivo del proyecto es poner en práctica los conocimientos adquiridos sobre JavaScript como:

Manejo de arrays
Manipulación del DOM entre JS y HTML
Función y validación de inputs
Generacion de números aleatorios
Funcionalidades

Agregar amigos: Permite ingresar nombres en un campo de texto y agregarlos a la lista.
Sortear un amigo secreto: Selecciona aleatoriamente un nombre del array y lo muestra en la pantalla
Validaciones: No se puede agregar un campo vacio, ni sortear si no hay nommmbres ingresados, muestra un mensaje de alerta.
Tectonolías

HTML5
CSS
JavaScript
Estructura del proyecto

/amigo-secreto ├── index.html # Estructura de la página ├── style.css # Estilos visuales ├── app.js # Lógica en JavaScript └── assets/ # Imágenes y recursos

Cómo usar

Abrir el archivo index.html en un navegador.
Escribir el nombre de un amigo en el campo de texcto y hacer clic en Añadir.
Repetir el paso anterior para todos los participantes.
Presionar Sortear Am igo para obtener un nommbre aleatorio de la lista.
Visualizar el resultado justo arriba del boton de sortear amigo.
